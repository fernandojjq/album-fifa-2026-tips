# FIFA 2026 Album Tips

A comprehensive guide website for collectors of the Panini FIFA 2026 World Cup sticker album.

## Features

- Complete album guide with tips and strategies
- Trading safety tips
- SWAP26 app recommendations
- Mobile-responsive design
- Multi-language support

## Pages

- **Home** (`/`) - Main landing page with tips and featured articles
- **Album Guide** (`/guide/`) - Complete guide to the Panini FIFA 2026 album
- **Collection Tips** (`/tips/`) - 15 tips to complete your album fast
- **Trading Tips** (`/trading/`) - How to trade stickers safely
- **SWAP26 App** (`/app/`) - Best app to manage your collection

## Getting Started

Simply open `index.html` in a web browser, or deploy to any static hosting service.

## Deployment

This is a static HTML website. Deploy to:
- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages
- Any web server

## Tech Stack

- HTML5
- CSS3 (with CSS variables)
- Vanilla JavaScript
- Google Fonts (Bebas Neue, Inter, Playfair Display)
- SVG icons
